On Git
--------

Generating patch: 
git format-patch <branch> --stdout > patch.txt 

Check the patch: 
git apply --stat patch.txt 

Dry run 
git apply --check patch.txt 

Applying the patch. There are two ways 
git am --signoff < patch.txt
